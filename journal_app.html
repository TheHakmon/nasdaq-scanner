כדי לראות את היומן מכל מכשיר, צור חשבון חינמי ב-jsonbin.io:
1. הירשם ב-jsonbin.io → Create Bin (תוכן התחלתי: [])
2. העתק את ה-BIN ID (מה-URL של ה-bin)
3. ב-API Keys העתק את ה-X-Master-Key
הדבק כאן - העסקאות יסתנכרנו אוטומטית בכל המכשירים שתגדיר בהם את אותם פרטים.
